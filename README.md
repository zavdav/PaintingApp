# JavaFX PaintingApp
![Demo image](demo.png)
## Features
### 1. Drawing Tools
* Freehand drawing/erasing
* Paint bucket area filling
* Drawing lines, ellipses & rectangles
### 2. Color selection
* RGB color selection
* Select color by eyedropper
### 3. Canvas
* Undo/redo changes
* Adjust canvas size
* Zoom in/out of canvas
### 4. Images
* Create, open & save images (.png)
